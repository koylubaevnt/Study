Input styles {{myinput.className}}

Обратите внимание на изменения классов input, при следующих действиях

  • Клике на поле ввода, потом за его пределами (ng-untouched -> ng-touched)
  • Вводе значения в пое ввода (ng-pristine -> ng-dirty)
  • Удаления значения из проля ввода (ng-valid -> ng-invalid)
